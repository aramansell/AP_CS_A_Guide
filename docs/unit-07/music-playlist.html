Music Playlist — Full Walkthrough

This is the program your teacher builds live. It uses ArrayList to manage a dynamic collection of songs — add, remove, search, shuffle, and display. Use this page as a reference.

The Goal

Build a playlist manager using ArrayList. Add songs, remove by title, search by artist, shuffle, and display the playlist.

=== MY PLAYLIST ===
1. "Bohemian Rhapsody" by Queen (5:55)
2. "Stairway to Heaven" by Led Zeppelin (8:02)
3. "Hotel California" by Eagles (6:30)
---
Total songs: 3
Total duration: 20:27
---
Search "Queen": Bohemian Rhapsody

Step 1: The Song Class

Small, focused class — just three fields and toString.

public class Song {
    private String title;
    private String artist;
    private int durationSeconds;

    public Song(String title, String artist, int durationSeconds) {
        this.title = title;
        this.artist = artist;
        this.durationSeconds = durationSeconds;
    }

    public String getTitle()  { return title; }
    public String getArtist() { return artist; }
    public int getDuration()  { return durationSeconds; }

    public String toString() {
        int min = durationSeconds / 60;
        int sec = durationSeconds % 60;
        return """ + title + "" by " + artist
               + " (" + min + ":" + String.format("%02d", sec) + ")";
    }
}

Step 2: ArrayList Basics — Add, Remove, Display

import java.util.ArrayList;

ArrayList<Song> playlist = new ArrayList<>();

playlist.add(new Song("Bohemian Rhapsody", "Queen", 355));
playlist.add(new Song("Stairway to Heaven", "Led Zeppelin", 482));
playlist.add(new Song("Hotel California", "Eagles", 390));

// Remove by title (case-insensitive)
String toRemove = "hotel california";
for (int i = 0; i < playlist.size(); i++) {
    if (playlist.get(i).getTitle().equalsIgnoreCase(toRemove)) {
        playlist.remove(i);
        break;
    }
}

// Display
for (int i = 0; i < playlist.size(); i++) {
    System.out.println((i + 1) + ". " + playlist.get(i));
}

Why ArrayList and not an array? The playlist size changes when songs are added or removed. ArrayList handles resizing automatically. No need to track count or shift elements by hand.

Step 3: Search and Shuffle

// Search by artist
public static void searchByArtist(ArrayList<Song> list, String artist) {
    boolean found = false;
    for (Song s : list) {
        if (s.getArtist().equalsIgnoreCase(artist)) {
            System.out.println("  " + s.getTitle());
            found = true;
        }
    }
    if (!found) System.out.println("  No songs found by " + artist);
}

// Shuffle (Fisher-Yates on ArrayList)
public static void shuffle(ArrayList<Song> list) {
    for (int i = list.size() - 1; i > 0; i--) {
        int j = (int) (Math.random() * (i + 1));
        Song temp = list.get(i);
        list.set(i, list.get(j));
        list.set(j, temp);
    }
}

The Full Program

import java.util.ArrayList;

public class PlaylistManager {
    public static void main(String[] args) {
        ArrayList<Song> playlist = new ArrayList<>();

        playlist.add(new Song("Bohemian Rhapsody", "Queen", 355));
        playlist.add(new Song("Stairway to Heaven", "Led Zeppelin", 482));
        playlist.add(new Song("Hotel California", "Eagles", 390));

        // Display
        System.out.println("=== MY PLAYLIST ===");
        int totalSec = 0;
        for (int i = 0; i < playlist.size(); i++) {
            System.out.println((i + 1) + ". " + playlist.get(i));
            totalSec += playlist.get(i).getDuration();
        }

        System.out.println("Total songs: " + playlist.size());
        System.out.println("Total duration: "
            + (totalSec / 60) + ":" + String.format("%02d", totalSec % 60));

        // Search
        System.out.println("
Search for "Queen":");
        for (Song s : playlist) {
            if (s.getArtist().equalsIgnoreCase("Queen")) {
                System.out.println("  " + s.getTitle());
            }
        }
    }
}
