Leaderboard — Full Walkthrough

This is the program your teacher builds live. It uses ArrayList with a ScoreEntry class to build a dynamic leaderboard that sorts, inserts, and trims. Use this page as a reference.

The Goal

Build a leaderboard that accepts any number of scores, keeps them sorted, and displays a top-N list. Unlike the Unit 6 fixed-size version, this grows dynamically.

=== LEADERBOARD (Top 5) ===
1. AAA — 9500
2. FFF — 6300
3. BBB — 7200
4. CCC — 5100
5. DDD — 3400

Step 1: ScoreEntry Class

public class ScoreEntry {
    private String player;
    private int score;

    public ScoreEntry(String player, int score) {
        this.player = player;
        this.score = score;
    }

    public String getPlayer() { return player; }
    public int getScore()     { return score; }

    public String toString() {
        return player + " — " + score;
    }
}

Step 2: ArrayList + Sorted Insert

ArrayList<ScoreEntry> leaderboard = new ArrayList<>();

public void addScore(String player, int score) {
    ScoreEntry newEntry = new ScoreEntry(player, score);

    // Find insertion point (keep highest first)
    int i = 0;
    while (i < leaderboard.size() && leaderboard.get(i).getScore() > score) {
        i++;
    }
    leaderboard.add(i, newEntry);  // ArrayList handles the shift!
}

add(index, element) does the shift for you. Unlike arrays (where you shift by hand), ArrayList.add(i, e) pushes everything from index i forward and inserts. Clean and readable.

Step 3: Display Top N

public void displayTop(int n) {
    System.out.println("=== LEADERBOARD (Top " + n + ") ===");
    int limit = Math.min(n, leaderboard.size());
    for (int i = 0; i < limit; i++) {
        System.out.println((i + 1) + ". " + leaderboard.get(i));
    }
    if (leaderboard.isEmpty()) {
        System.out.println("(No scores yet)");
    }
}

The Full Program

import java.util.ArrayList;

public class Leaderboard {
    private ArrayList<ScoreEntry> entries;

    public Leaderboard() {
        entries = new ArrayList<>();
    }

    public void addScore(String player, int score) {
        ScoreEntry entry = new ScoreEntry(player, score);
        int i = 0;
        while (i < entries.size() && entries.get(i).getScore() > score) {
            i++;
        }
        entries.add(i, entry);
    }

    public void displayTop(int n) {
        System.out.println("=== LEADERBOARD (Top " + n + ") ===");
        int limit = Math.min(n, entries.size());
        for (int i = 0; i < limit; i++) {
            System.out.println((i + 1) + ". " + entries.get(i));
        }
    }

    public static void main(String[] args) {
        Leaderboard lb = new Leaderboard();
        lb.addScore("AAA", 9500);
        lb.addScore("BBB", 7200);
        lb.addScore("FFF", 6300);
        lb.addScore("CCC", 5100);
        lb.addScore("DDD", 3400);
        lb.addScore("EEE", 1200);
        lb.displayTop(5);
    }
}

Extending the Program

  • Prevent duplicate player names — keep only the highest score per player.
  • Use Collections.sort() with a custom Comparator instead of sorted insert.
  • Export to CSV for a spreadsheet.

How This Connects to Your RPG

The sorted insert is the same logic as adding items to an inventory sorted by value or rarity. ArrayList.add(index, element) replaces your manual array-shift code from Unit 6.
