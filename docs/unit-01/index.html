Unit 1: Primitive Types

Every program stores data. This unit is about the simplest kinds of data Java gives you: whole numbers, decimals, and true/false. You'll also learn to print things, do math, and generate random numbers — the foundation of every dice roll, damage calculation, and loot drop in your RPG.

Concepts

TopicWhat you'll learn
Compile & Run How to turn .java into a running program. The golden rule.
Variables & Types int, double, boolean — declaring, initializing, naming.
Arithmetic & Operators +, -, *, /, %, and the integer division trap.
Math.random() Generating random numbers. The dice-rolling engine.
Printing & Concatenation println, print, and the + operator with strings.
The Three-Step Pattern Declare → Calculate → Display. The structure of nearly every program you'll write.
