Slot Machine — Full Walkthrough

This is the program your teacher built live. It uses variables, arithmetic, printing, Math.random(), and the three-step pattern.

The Goal

Simulate a three-reel slot machine. Generate three random symbols, display them, and announce a winner if all three match.

=== SLOT MACHINE ===
Spinning...
[ Cherry | Lemon | Cherry ]
No match — try again!

Iteration 1: The Broken Reels

Below is an early version of the code we wrote in class. Copy this into a file named SlotMachine.java and run it. You will notice a major bug.

public class SlotMachine {
    public static void main(String[] args) {
        // Step 1: Declare
        int reel1;
        int reel2;
        int reel3;

        // Step 2: Generate
        reel1 = (int) Math.random() * 4;
        reel2 = (int) Math.random() * 4;
        reel3 = (int) Math.random() * 4;

        // Step 3: Map to Symbols (Simplified for testing)
        // Let's just print the numbers first to see if they work
        System.out.println("=== SLOT MACHINE ===");
        System.out.println("[ " + reel1 + " | " + reel2 + " | " + reel3 + " ]");
    }
}

Bug Hunt 1:

Run the code 5 times. What is the output every single time? Why? Hint: Look closely at the casting on the generation lines. Refer back to the Math.random() documentation if you are stuck.

Fix the bug in your code before moving on.

Iteration 2: Adding the Symbols

Once the reels are generating random numbers from 0 to 3, we need to map them to Strings. Here is the logic we added for the first reel:

        String symbol1;
        if (reel1 == 0) symbol1 = "Cherry";
        else if (reel1 == 1) symbol1 = "Lemon";
        else if (reel1 == 2) symbol1 = "7";
        else symbol1 = "Bell";

Tinker Challenge:

You need to repeat this logic for reel2 (saving into symbol2) and reel3 (saving into symbol3). Add this code, and update your print statement at the end to print the Strings instead of the integers.

Iteration 3: The Win Condition

Finally, we need to check if the player won. Here is the code with a logic error.

        if (reel1 == reel2) {
            System.out.println("JACKPOT! All three match!");

Bug Hunt 2:

What is wrong with the win condition above? (Think about a scenario where reel1 is Cherry, reel2 is Cherry, and reel3 is Lemon). How do you modify the `if` statement to ensure ALL THREE match?

Extending the Program (Mandatory Tinker Challenges)

Once your slot machine works perfectly, pick at least one of these to implement:

  • Change the random generation to * 6 and add two more symbols (e.g., "Diamond" and "Star").
  • Add a payout system: Start the player with 100 coins. If they win, add 50 coins. Print their final balance at the end.
  • Give two-of-a-kind a "small payout" message.
