Choose Your Own Adventure — Full Walkthrough

This is the program your teacher builds live. It uses nested if/else to create branching story paths. Use this page as a reference when building your own projects.

The Goal

Present the player with choices. Each choice leads to a different branch of the story. Use nested if/else to build a tree of possibilities.

You enter a dark dungeon. Two tunnels stretch ahead.
Do you go LEFT or RIGHT? left

You go left. The tunnel opens into a treasure room!
A chest sits in the corner. Do you OPEN it or LEAVE it? open

You open the chest and find 100 gold coins!
Your adventure ends... for now.

Step 1: Read the First Choice

import java.util.Scanner;

Scanner input = new Scanner(System.in);

System.out.println("You enter a dark dungeon. Two tunnels stretch ahead.");
System.out.println("Do you go LEFT or RIGHT?");
String choice1 = input.nextLine().toLowerCase();

Step 2: Branch Based on First Choice

if (choice1.equals("left")) {
    System.out.println("You go left. The tunnel opens into a treasure room!");
    // Second choice inside this branch

} else if (choice1.equals("right")) {
    System.out.println("You go right. A dragon blocks the path!");
    // Different second choice here

} else {
    System.out.println("Paralyzed by indecision, you turn back.");
}

Nesting: Each branch is its own mini-adventure. Put more if/else inside each branch to give the player a second choice — and a third, and a fourth. This is the same structure as a "choose your own adventure" book.

Step 3: Add Second and Third Layers

if (choice1.equals("left")) {
    System.out.println("You go left. The tunnel opens into a treasure room!");
    System.out.println("A chest sits in the corner. Do you OPEN it or LEAVE it?");
    String choice2 = input.nextLine().toLowerCase();

    if (choice2.equals("open")) {
        System.out.println("You open the chest and find 100 gold coins!");
        // Could add a THIRD layer here...

    } else if (choice2.equals("leave")) {
        System.out.println("You leave the chest. A trap avoids you. Wise choice.");
    }
}

The Full Program

import java.util.Scanner;

public class ChooseAdventure {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        System.out.println("You enter a dark dungeon. Two tunnels stretch ahead.");
        System.out.print("Do you go LEFT or RIGHT? ");
        String choice1 = input.nextLine().toLowerCase();

        if (choice1.equals("left")) {
            System.out.println("\nYou enter a treasure room!");
            System.out.print("A chest sits in the corner. OPEN it or LEAVE it? ");
            String choice2 = input.nextLine().toLowerCase();

            if (choice2.equals("open")) {
                System.out.println("\nYou open the chest. 100 gold coins!");
                System.out.println("A passage opens behind the chest...");
                System.out.print("GO through it or RETURN? ");
                String choice3 = input.nextLine().toLowerCase();

                if (choice3.equals("go")) {
                    System.out.println("\nYou emerge into sunlight. Freedom!");
                } else {
                    System.out.println("\nYou return. The chest was the best part anyway.");
                }
            } else {
                System.out.println("\nYou leave the chest. A trap clicks harmlessly.");
                System.out.println("You exit safely with nothing... but alive.");
            }
        } else if (choice1.equals("right")) {
            System.out.println("\nA dragon blocks the path!");
            System.out.print("FIGHT it or RUN? ");
            String choice2 = input.nextLine().toLowerCase();

            if (choice2.equals("fight")) {
                System.out.println("\nYou charge! The dragon breathes fire.");
                System.out.println("GAME OVER — you're toast.");
            } else {
                System.out.println("\nYou run back to the entrance. Safety first.");
            }
        } else {
            System.out.println("\nYou stand frozen. The ceiling collapses.");
            System.out.println("GAME OVER — indecision kills.");
        }

        System.out.println("Your adventure ends... for now.");
    }
}

Extending the Program

  • Draw a story map on paper first — a tree diagram helps.
  • Add more than two choices at each branch (if / else-if / else).
  • Reuse branches: two different paths could lead to the same ending.

How This Connects to Your RPG

Your Combat System uses the same if/else chain to determine hit/miss/crit. Choose Your Adventure stretches that chain into a tree. The RPG's "dungeon crawl" structure is essentially a Choose Your Adventure with combat mixed in.
