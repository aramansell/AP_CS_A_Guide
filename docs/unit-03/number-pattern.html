Number Pattern Generator — Full Walkthrough

This is the program your teacher builds live. It uses if/else to classify numbers and prints formatted patterns. Use this page as a reference when building your own projects.

The Goal

Build three mini-programs on one page: FizzBuzz, a multiplication table snippet, and a star pattern — each controlled by conditionals.

Part 1: FizzBuzz Logic

int num = 15;

if (num % 3 == 0 && num % 5 == 0) {
    System.out.println("FizzBuzz!");
} else if (num % 3 == 0) {
    System.out.println("Fizz");
} else if (num % 5 == 0) {
    System.out.println("Buzz");
} else {
    System.out.println(num);
}

Bug Hunt: FizzBuzz Order

A student wrote their FizzBuzz like this:

if (num % 3 == 0) {
    System.out.println("Fizz");
} else if (num % 5 == 0) {
    System.out.println("Buzz");
} else if (num % 3 == 0 && num % 5 == 0) {
    System.out.println("FizzBuzz!");
} else {
    System.out.println(num);
}

When they test it with 15, it prints "Fizz". It NEVER prints "FizzBuzz!".

  1. Why does this happen? (Trace the execution top-to-bottom).
  2. What is the rule about ordering conditions from most specific to most general?

Part 2: Multiplication Table

int factor = 5;

for (int i = 1; i <= 10; i++) {
    System.out.println(factor + " x " + i + " = " + (factor * i));
}

This is a for loop. Covered formally in Unit 4, but the pattern is useful now. The loop repeats 10 times with i counting 1..10.

Part 3: Star Pattern

for (int row = 1; row <= 5; row++) {
    String line = "";
    for (int col = 1; col <= row; col++) {
        line = line + "*";
    }
    System.out.println(line);
}

Socratic Checkpoint: Nested Loops

Look at the two loops. For every 1 time the outer loop runs, how many times does the inner loop run?

When row is 3, what is the value of col during each iteration of the inner loop? What will the line string contain at the end of that row?

The Full Program

import java.util.Scanner;

public class NumberPatterns {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        // === PART 1: FIZZBUZZ ===
        System.out.println("=== FIZZBUZZ ===");
        System.out.print("Enter a number: ");
        int num = input.nextInt();

        if (num % 3 == 0 && num % 5 == 0) {
            System.out.println("FizzBuzz!");
        } else if (num % 3 == 0) {
            System.out.println("Fizz");
        } else if (num % 5 == 0) {
            System.out.println("Buzz");
        } else {
            System.out.println(num);
        }

        // === PART 2: MULTIPLICATION TABLE ===
        System.out.println("
=== MULTIPLICATION TABLE ===");
        System.out.print("Enter a factor: ");
        int factor = input.nextInt();

        for (int i = 1; i <= 10; i++) {
            int product = factor * i;
            System.out.println(factor + " x " + i + " = " + product);
        }

        // === PART 3: STAR TRIANGLE ===
        System.out.println("
=== STAR TRIANGLE ===");
        System.out.print("Enter height: ");
        int height = input.nextInt();

        for (int row = 1; row <= height; row++) {
            String line = "";
            for (int col = 1; col <= row; col++) {
                line += "*";
            }
            System.out.println(line);
        }
    }
}

Extending the Program
