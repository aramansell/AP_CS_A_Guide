Number Guessing Game — Full Walkthrough

This is the program your teacher builds live. It uses conditionals and comparison operators to give the player feedback. Use this page as a reference when building your own projects.

The Goal

Generate a secret number (1–100), read the player's guess, and tell them if they're too high, too low, or correct.

I'm thinking of a number between 1 and 100.
Your guess: 50
Too high! Guess lower.
Your guess: 25
Too low! Guess higher.
Your guess: 37
Correct! The number was 37.

Step 1: Generate the Secret Number

int secret = (int) (Math.random() * 100) + 1;

Why + 1? Math.random() * 100 gives [0, 100). Cast to int: 0–99. Add 1: 1–100. Exactly the range we want.

Step 2: Read the Guess

Scanner input = new Scanner(System.in);

System.out.println("I'm thinking of a number between 1 and 100.");
System.out.print("Your guess: ");
int guess = input.nextInt();

Why nextInt() and not nextLine()? The guess is a number. nextInt() reads an int directly — no need to parse with Integer.parseInt(). Cleaner code.

Step 3: Give Feedback with If/Else

if (guess > secret) {
    System.out.println("Too high! Guess lower.");
} else if (guess < secret) {
    System.out.println("Too low! Guess higher.");
} else {
    System.out.println("Correct! The number was " + secret + ".");
}

Socratic Checkpoint: Mutual Exclusivity

Look at the code above. Why did we use else if and else instead of three separate if statements?

What would happen if we used three independent if statements and the user guessed the exact secret number? Trace the logic to find out.

The Full Program

import java.util.Scanner;

public class NumberGuessing {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        // ===== STEP 1: GENERATE SECRET =====
        int secret = (int) (Math.random() * 100) + 1;

        // ===== STEP 2: GET GUESS =====
        System.out.println("I'm thinking of a number between 1 and 100.");
        System.out.print("Your guess: ");
        int guess = input.nextInt();

        // ===== STEP 3: FEEDBACK =====
        if (guess > secret) {
            System.out.println("Too high! Guess lower.");
        } else if (guess < secret) {
            System.out.println("Too low! Guess higher.");
        } else {
            System.out.println("Correct! The number was " + secret + ".");
        }
    }
}

From One Guess to Many

One guess is a fine start. But a real guessing game lets the player keep trying. In Unit 4, you'll wrap this in a while loop so the player guesses until they get it right. The conditional logic stays exactly the same — you just put it inside a loop.

Preview — this is what Unit 4 will enable:

while (guess != secret) {
    if (guess > secret) {
        System.out.print("Too high! Try again: ");
    } else {
        System.out.print("Too low! Try again: ");
    }
    guess = input.nextInt();
}
System.out.println("Correct! The number was " + secret + ".");

Extending the Program

  • Track the number of guesses and display it at the end.
  • Add difficulty levels: easy (1–50), medium (1–100), hard (1–500).
  • Give "warm" (within 10) and "cold" (more than 20 away) hints.
