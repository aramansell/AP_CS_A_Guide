Tracing Loops

Trace tables let you simulate a loop on paper — you track every variable at every step. This is the skill that separates students who guess from students who know.

What Is a Trace Table?

A trace table is a grid. Each row is one iteration. Each column is a variable.

// Code to trace:
int sum = 0;
for (int i = 1; i <= 4; i++) {
    sum += i;
}
System.out.println("sum = " + sum);

The Off-By-One Error

This is the single most common loop bug. The loop runs one too many or one too few times. 

// BUG: runs 6 times, not 5
for (int i = 0; i <= 5; i++) { ... }

// BUG: runs 0 times (condition starts false)
for (int i = 5; i < 5; i++) { ... }

// BUG: infinite — i never changes
for (int i = 0; i < 5;) { ... }

// BUG: runs 5 times but i goes 1,2,3,4,5 — not 0,1,2,3,4
for (int i = 1; i <= 5; i++) { ... }

🐛 Bug Hunt: Off-By-One

A student wants a loop that runs exactly 5 times, but they wrote:

for (int i = 0; i <= 5; i++) {
    System.out.println("Iteration: " + i);
}

Your Task: Hand-trace the first and last iteration. How many times does this loop actually run? What exact change would fix the bug?

🧠 Socratic Checkpoint: Exam Prep

AP-style questions often ask you to trace loops or identify equivalent loops without running the code.

Consider this loop:

for (int i = 0; i < 5; i++) { System.out.print(i + " "); }
// Output: 0 1 2 3 4

Think about it: Which of the following loops produces the EXACT same output, and why?

  • (a) for (int i = 1; i < 5; i++)
  • (b) for (int i = 1; i <= 5; i++)
  • (c) for (int i = 0; i <= 4; i++)
  • (d) for (int i = 0; i < 6; i++)

🛠️ Tinker Challenge: Trace and Predict

Practice tracing loops on paper (or a whiteboard) before running the code.

  • Trace this loop on paper: for (int i = 0; i < 5; i++) { sum += i * 2; }. What is sum?
  • Find the bug in this countdown: for (int i = 10; i > 0; i++) { ... }
  • Write a for loop that prints 3, 6, 9, 12, 15 without hardcoding the numbers directly into the print statement.
  • Bonus: Trace a while loop from your Combat.java code. How many iterations does it take for a 20 HP player to defeat a 12 HP goblin if both deal average damage?
