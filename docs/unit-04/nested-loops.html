Nested Loops

A nested loop is a loop inside another loop. For each iteration of the outer loop, the inner loop runs to completion. This is how you process grids, tables, and multi-dimensional data.

The Pattern

for (int row = 1; row <= 3; row++) {         // OUTER: runs 3 times
    for (int col = 1; col <= 4; col++) {     // INNER: runs 4 times PER outer
        System.out.print("(" + row + "," + col + ") ");
    }
    System.out.println();  // newline after each row
}

(1,1) (1,2) (1,3) (1,4)
(2,1) (2,2) (2,3) (2,4)
(3,1) (3,2) (3,3) (3,4)

The outer loop runs 3 times. Each time, the inner loop runs 4 times. Total iterations: 3 × 4 = 12.

Multiplication Table

// Print a 10x10 multiplication table
for (int i = 1; i <= 10; i++) {
    for (int j = 1; j <= 10; j++) {
        System.out.print(i * j + "\t");
    }
    System.out.println();
}

Application: Dungeon Grid

// Map a dungeon: 5 rows, 5 columns
// "." = empty room, "E" = enemy, "T" = treasure, "B" = boss

char[][] dungeon = {
    {'.', '.', 'E', '.', '.'},
    {'.', 'T', '.', 'E', '.'},
    {'E', '.', '.', '.', 'T'},
    {'.', '.', 'E', '.', '.'},
    {'.', '.', '.', 'B', '.'}
};

System.out.println("=== DUNGEON MAP ===");
for (int row = 0; row < dungeon.length; row++) {
    for (int col = 0; col < dungeon[row].length; col++) {
        System.out.print(dungeon[row][col] + " ");
    }
    System.out.println();
}

=== DUNGEON MAP ===
. . E . .
. T . E .
E . . . T
. . E . .
. . . B .

Application: Combat with Multiple Enemies

String[] enemies = {"Goblin", "Skeleton", "Orc"};

for (int round = 1; round <= 3; round++) {
    System.out.println("--- Round " + round + " ---");

    for (int i = 0; i < enemies.length; i++) {
        int roll = (int)(Math.random() * 20) + 1;
        if (roll >= 10) {
            System.out.println("  " + enemies[i] + " attacks!");
        } else {
            System.out.println("  " + enemies[i] + " hesitates...");
        }
    }
}

Triangle Patterns

// Right triangle of dungeon levels
for (int row = 1; row <= 5; row++) {
    for (int col = 1; col <= row; col++) {
        System.out.print("█");
    }
    System.out.println("  — Level " + row);
}

Analysis: How Many Times?

For nested loops, multiply the iterations:

for (int i = 0; i < n; i++) {       // n times
    for (int j = 0; j < m; j++) {   // m times per i
        // body runs n × m times
    }
}

When to Use Nested Loops

  • Grids and maps — each cell (row, col) gets processed
  • Tables — every combination of row and column header
  • Combat rounds with multiple enemies — outer: rounds, inner: enemies
  • Dungeon exploration — outer: rooms, inner: actions per room
  • Pattern printing — shapes, ASCII art, dungeon maps
