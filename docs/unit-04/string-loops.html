Loops + Strings

Strings are sequences of characters. Loops let you visit every character one at a time — string traversal. You can search, count, transform, and validate text.

Accessing Characters: charAt()

charAt(index) returns the character at that position. Index starts at 0.

String weapon = "Longsword";
// Index:          012345678

weapon.charAt(0);  // 'L'
weapon.charAt(4);  // 's'
weapon.charAt(8);  // 'd'

🐛 Bug Hunt: Out of Bounds

A student wrote this code to print the last letter of a weapon's name:

String weapon = "Longsword";
char lastLetter = weapon.charAt(weapon.length());
System.out.println(lastLetter);

String Traversal Pattern

String spell = "Fireball";

for (int i = 0; i < spell.length(); i++) {
    char c = spell.charAt(i);
    System.out.println("Position " + i + ": " + c);
}

Position 0: F
Position 1: i
Position 2: r
Position 3: e
Position 4: b
Position 5: a
Position 6: l
Position 7: l

Counting Characters

// How many 'l's in "Fireball"?
String spell = "Fireball";
int count = 0;

for (int i = 0; i < spell.length(); i++) {
    if (spell.charAt(i) == 'l') {
        count++;
    }
}
System.out.println("Found " + count + " 'l' characters.");
// → "Found 2 'l' characters."

Building Strings in a Loop

Strings are immutable — you can't change them. Instead, build a new one. 

// Reverse a string
String original = "sword";
String reversed = "";

for (int i = original.length() - 1; i >= 0; i--) {
    reversed += original.charAt(i);
}
System.out.println(reversed);  // "drows"

🧠 Socratic Checkpoint: The Memory Cost

Strings in Java are immutable, meaning they can never be changed once created. Every time you use +=, Java creates a brand new String and throws the old one away.

Think about it: If you use a loop to build a String of 1,000 characters one by one using +=, how many temporary Strings does Java have to create and destroy? (Hint: Later, you'll learn about StringBuilder which solves this problem!)

Application: RPG Word Filter

// Check if a spell name starts with "Fire"
String spell = "Fireball";

if (spell.length() >= 4) {
    String prefix = "";
    for (int i = 0; i < 4; i++) {
        prefix += spell.charAt(i);
    }
    if (prefix.equals("Fire")) {
        System.out.println("FIRE spell detected! Boost damage.");
    }
}

Easier way (once you know substring):

if (spell.substring(0, 4).equals("Fire")) { ... }

Finding Substrings

// Does the password contain "dragon"?
String password = "red-dragon-gate";
String keyword = "dragon";
boolean found = false;

for (int i = 0; i <= password.length() - keyword.length(); i++) {
    String piece = password.substring(i, i + keyword.length());
    if (piece.equals(keyword)) {
        found = true;
        break;  // stop — found it
    }
}

Application: Combat Command Parser

Scanner input = new Scanner(System.in);
System.out.print("Action: ");
String cmd = input.nextLine();

// Parse "attack goblin" into verb and target
int spacePos = -1;
for (int i = 0; i < cmd.length(); i++) {
    if (cmd.charAt(i) == ' ') {
        spacePos = i;
        break;
    }
}

if (spacePos == -1) {
    System.out.println("One-word command: " + cmd);
} else {
    String verb = cmd.substring(0, spacePos);
    String target = cmd.substring(spacePos + 1);
    System.out.println("Verb: \"" + verb + "\" → Target: \"" + target + "\"");
}

Character Testing Methods

These Character class methods test what kind of character something is:

MethodTrue for...
Character.isDigit(c)'0'–'9'
Character.isLetter(c)'a'–'z', 'A'–'Z'
Character.isUpperCase(c)'A'–'Z'
Character.isLowerCase(c)'a'–'z'
Character.isWhitespace(c)space, tab, newline
// Count only the letters in a player name
String name = "Gandalf99";
int letters = 0;
for (int i = 0; i < name.length(); i++) {
    if (Character.isLetter(name.charAt(i))) {
        letters++;
    }
}
