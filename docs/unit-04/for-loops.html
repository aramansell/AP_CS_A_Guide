for Loops

A for loop is a count-controlled loop. Use it when you know exactly how many times something should repeat — like "do this for every room" or "roll damage five times."

Syntax

for (initialize; test; update) {
    // body
}

All three parts of a while loop, packed into one line:

//     initialize     test       update
//       ↓             ↓          ↓
for (int i = 0;    i < 5;       i++) {
    System.out.println("Loop #" + i);
}

How It Executes

  1. Initialize — runs ONCE at the very start.
  2. Test — if true, run body. If false, stop.
  3. Body — the statements inside the braces.
  4. Update — runs AFTER the body. Then go back to step 2.

Counting Patterns

// 0 to 9 (10 times — standard array pattern)
for (int i = 0; i < 10; i++) { ... }

// 1 to 10
for (int i = 1; i <= 10; i++) { ... }

// 10 down to 1
for (int i = 10; i >= 1; i--) { ... }

// Even numbers: 0, 2, 4, 6, 8
for (int i = 0; i < 10; i += 2) { ... }

// 10, 20, 30, ..., 100
for (int i = 10; i <= 100; i += 10) { ... }

Loop Variable Scope

A variable declared in the for-loop header only exists inside the loop.

🧠 Socratic Checkpoint: Lost in Scope

Look at this code snippet:

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
System.out.println("The final value is: " + i);

Think about it: Why does the final println cause a compiler error? If you really needed to know the final value of i after the loop, how would you rewrite this code to make it work?

Application: Inventory Display

String[] items = {"Torch", "Sword", "Potion", "Shield", "Key"};

System.out.println("=== INVENTORY ===");
for (int i = 0; i < items.length; i++) {
    System.out.println((i + 1) + ". " + items[i]);

Enhanced for Loop (Preview)

When you don't need the index, there's a shorter syntax. This is a preview — you'll use it more in later units: 

for (String item : items) {
    System.out.println(item);
}

🛠️ Tinker Challenge: The Gauntlet

Write a for loop that generates exactly 5 consecutive rooms in a dungeon.

  • For each room, print its number (e.g., "--- Room 1 ---").
  • Inside the loop, roll a d20 to determine the enemy:
    • 1-10: Goblin (10 HP)
    • 11-18: Skeleton (15 HP)
    • 19-20: Orc (25 HP)
  • Print the enemy that appears and its HP.

Bonus: Keep a running total of the combined HP of all 5 monsters and print it at the very end!
