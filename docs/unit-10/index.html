Unit 10: Recursion

A method that calls itself. The most mind-bending idea in the course. Once you get it, you feel unstoppable.

This unit has 8 days: 3 teacher live-codes, 3 navigator builds, and 2 build days. The RPG gains fireball chain spells and dungeon torch lighting.

Concepts

TopicWhat you'll learn
Recursion Basics A method calling itself. Base case + recursive call. The call stack.
Recursive Search Searching arrays without loops.
Fibonacci & Efficiency When recursion gets expensive. Memoization as a fix.

Teacher Projects (Standalone Demos)

ProjectWhat it demonstrates
Recursion Basics Countdown, factorial, and the call stack. Base case + recursive step pattern.
Recursive Search Linear and binary search without loops. Divide-and-conquer thinking.
Fibonacci & Efficiency Exponential blowup, overlapping subproblems, memoization as the fix.

Your RPG Project

GuideWhat you'll build
Chain Lightning (Student RPG Guide) Recursive AoE spell that chains through enemies. Visited tracking.
Flood Fill / Fireball (Student RPG Guide) Hits every enemy in a connected region. Same damage to all.
Pathfinding (Student RPG Guide) Backtracking algorithm for dungeon navigation. Monster chase AI.

RPG Thread

Everything stacks. Chain lightning uses GameCharacter from Unit 9 on the grid from Unit 8. Flood fill tests whether torchlight reaches every connected room. Backtracking gives your dungeon AI a brain — monsters that actually chase the player.

Prerequisites

Before starting this unit, you should be comfortable with:

  • Inheritance and polymorphism: Unit 9
  • 2D arrays and grid traversal: Unit 8
  • ArrayLists and searching: Unit 7
  • Loops (while, for, nested): Unit 4

Reference Materials

  • Quick Reference: reference/RecursionQuickRef.md — base case patterns, call stack diagrams, common traps

Assessment

  • Part A: Knowledge — base case, recursive call, call stack, StackOverflowError
  • Part B: Code trace — predict output of recursive methods, trace the call stack
  • Part C: Write code — implement a recursive method from a specification
