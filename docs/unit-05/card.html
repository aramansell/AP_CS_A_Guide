Writing a Card Class — Full Walkthrough

This is the program your teacher builds live. You'll design a Card class from scratch — instance variables, constructors, getters, toString, and comparison logic. Use this as a reference when building your own classes.

The Goal

Create a Card class representing a playing card — suit, rank, and point value. Then test it by creating a few cards and comparing them.

Card c1 = new Card("Hearts", "Ace", 14);
Card c2 = new Card("Spades", "King", 13);
c1: Ace of Hearts (14 pts)
c2: King of Spades (13 pts)
c1 beats c2? true

Step 1: The Spec

Before writing code, define what the class needs:

  • Instance variables: suit (String), rank (String), pointValue (int)
  • Constructor: takes all three, sets them
  • Getters: getSuit(), getRank(), getPointValue()
  • toString(): "Ace of Hearts (14 pts)"
  • Extra: beats(Card other) — returns true if this card's points > other card's points

Step 2: Instance Variables + Constructor

public class Card {
    private String suit;
    private String rank;
    private int pointValue;

    public Card(String suit, String rank, int pointValue) {
        this.suit = suit;
        this.rank = rank;
        this.pointValue = pointValue;
    }

Why private? Suit and rank shouldn't change after the card is created. Private protects them. The card is immutable — a feature, not a bug.

Step 3: Getters and toString

    public String getSuit() {
        return suit;
    }

    public String getRank() {
        return rank;
    }

    public int getPointValue() {
        return pointValue;
    }

    public String toString() {
        return rank + " of " + suit + " (" + pointValue + " pts)";
    }

Step 4: Comparison Method

    public boolean beats(Card other) {
        return this.pointValue > other.pointValue;
    }
}

this vs other: The method is called on one card: card1.beats(card2). Inside the method, this refers to card1, and other is the parameter (card2).

The Full Class + Test

public class Card {
    private String suit;
    private String rank;
    private int pointValue;

    public Card(String suit, String rank, int pointValue) {
        this.suit = suit;
        this.rank = rank;
        this.pointValue = pointValue;
    }

    public String getSuit()  { return suit; }
    public String getRank()  { return rank; }
    public int getPointValue() { return pointValue; }

    public String toString() {
        return rank + " of " + suit + " (" + pointValue + " pts)";
    }

    public boolean beats(Card other) {
        return this.pointValue > other.pointValue;
    }
}

// Test in main():
public class CardTest {
    public static void main(String[] args) {
        Card ace = new Card("Hearts", "Ace", 14);
        Card king = new Card("Spades", "King", 13);

        System.out.println(ace);       // Ace of Hearts (14 pts)
        System.out.println(king);      // King of Spades (13 pts)
        System.out.println("Ace beats King? " + ace.beats(king));  // true
    }
}
