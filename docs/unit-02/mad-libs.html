Mad Libs — Full Walkthrough

This is the program your teacher builds live. It uses Scanner to read user input and concatenation to build a story. Use this page as a reference when building your own projects.

The Goal

Prompt the user for words (noun, verb, adjective, etc.), then plug them into a story template and display the result.

=== MAD LIBS ===
Enter a noun: dragon
Enter a verb ending in -ing: flying
Enter an adjective: enormous
Enter a place: castle
Enter a name: Bob
--- YOUR STORY ---
Bob saw an enormous dragon flying over the castle.

Why all Strings? Words are text. Scanner.nextLine() returns a String every time.

Step 2: Prompt and Read

For each word: print a prompt, read the answer. The pattern repeats.

System.out.print("Enter a noun: ");
noun = input.nextLine();

System.out.print("Enter a verb ending in -ing: ");
verb = input.nextLine();

System.out.print("Enter an adjective: ");
adjective = input.nextLine();

System.out.print("Enter a place: ");
place = input.nextLine();

System.out.print("Enter a name: ");
name = input.nextLine();

Why nextLine() and not next()? nextLine() reads the entire line including spaces. If the user types "fire-breathing dragon" for the noun, next() would only grab "fire-breathing". nextLine() gets the whole thing.

Test it. After each nextLine(), add a temporary print to confirm you got the word right. Then remove the test prints.

Bug Hunt: The Spacing Trap

Run the following code: 
System.out.println(name + "saw" + adjective + noun);

What does the output look like? Why did it run all the words together? How do you fix it using String concatenation?

The Full Program (For Reference)

import java.util.Scanner;

public class MadLibs {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        // ===== STEP 1: DECLARE =====
        String noun;
        String verb;
        String adjective;
        String place;
        String name;

        // ===== STEP 2: INPUT =====
        System.out.println("=== MAD LIBS ===");
        System.out.print("Enter a noun: ");
        noun = input.nextLine();

        System.out.print("Enter a verb ending in -ing: ");
        verb = input.nextLine();

        System.out.print("Enter an adjective: ");
        adjective = input.nextLine();

        System.out.print("Enter a place: ");
        place = input.nextLine();

        System.out.print("Enter a name: ");
        name = input.nextLine();

        // ===== STEP 3: DISPLAY =====
        System.out.println("--- YOUR STORY ---");
        System.out.println(name + " saw " + adjective + " " + noun
                           + " " + verb + " over the " + place + ".");
    }
}

Tinker Challenge: Extending the Story

  • Add an adverb ("clumsily"), a number ("three"), and a second adjective. Update the story to include them!
  • Write two or three story templates and let the user pick one (you might need to learn about `if` statements first, or just ask them for a number and use a math trick).
  • Use String methods like `toUpperCase()` to capitalize the first letter of the name or make the whole name shout.
