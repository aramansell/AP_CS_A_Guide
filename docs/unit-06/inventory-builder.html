Student RPG Guide: Item.java + FixedInventory.java

Two files. One system. Your RPG's inventory has been a single String array since Unit 2. Time for a real data model. Item.java is immutable (like Card). FixedInventory.java is a mutable container (like Deck). You build both from the spec.

References

  • Card (Unit 5) — immutable class, final fields, validation in constructor
  • BankAccount (Unit 5) — guard clauses, returning booleans for success/failure
  • CardDeck — array of objects, pointer-based dealing (same pattern as itemCount)
  • High Score Table — array shift for ordered insert/remove
  • Array Algorithms — shift-elements-left, search, sum traversal

FILE 1: Item.java

Immutable — like Card. Once created, an Item never changes.

Instance Variables

All private final.

VariableTypePurpose
nameStringItem name, e.g. "Steel Sword"
weightintEncumbrance, 1-50
goldValueintSell price, >= 0
rarityStringOne of: "Common", "Rare", "Epic", "Legendary"

Constructor + Validation

  • Takes four parameters: String name, int weight, int goldValue, String rarity
  • Validation rules (apply BEFORE storing):
    • name — if null or empty, set to "Unknown Item"
    • weight — clamp to [1, 50]
    • goldValue — clamp to >= 0. A free item is value 0.
    • rarity — must be exactly "Common", "Rare", "Epic", or "Legendary". Use .equals(). Default to "Common" if invalid.

Getters + toString()

  • Standard getters for all four fields
  • toString() format: "<name> (<rarity>) - Wt:<weight> Val:<value>g"

FILE 2: FixedInventory.java

Mutable container — like Deck. Backed by a fixed-size Item[] array with a itemCount pointer.

Instance Variables

VariableTypePurpose
itemsItem[]The storage array
itemCountintHow many items currently stored (0 to items.length)

Constructor

  • Parameter: int capacity
  • Allocate items = new Item[capacity]
  • itemCount starts at 0

Methods

MethodReturnsWhat It Does
add(Item item) boolean Add to items[itemCount], increment count. Return false if item is null OR inventory full. Return true on success.
remove(int index) Item Remove item at index. Shift all later items left to fill the gap. Clear the last slot to null. Return the removed item (or null if index invalid).
removeByName(String name) Item Find first item matching name (use .equalsIgnoreCase()). Call remove(index). Return the removed item or null.
contains(String name) boolean Loop. Check equalsIgnoreCase. Return true if found.
getTotalValue() int Sum goldValue of all items
getTotalWeight() int Sum weight of all items
countByRarity(String rarity) int Count how many items match the given rarity
display() void Print index + toString per item. Print "(empty)" if itemCount == 0.
getItemCount() int Return itemCount
isFull() boolean Returns itemCount == items.length

Test It

Put this in main() of FixedInventory.java or a separate runner:

// === Test 1: Item creation + validation ===
Item sword = new Item("Steel Sword", 10, 250, "Rare");
System.out.println(sword);
// Expected: "Steel Sword (Rare) - Wt:10 Val:250g"

Item junk = new Item("", -5, -100, "Legendaryyy");
System.out.println(junk);
// Expected: "Unknown Item (Common) - Wt:1 Val:0g"

// === Test 2: Inventory add ===
FixedInventory bag = new FixedInventory(5);
System.out.println(bag.add(sword));                  // true
System.out.println(bag.add(new Item("Shield", 15, 100, "Common"))); // true
System.out.println(bag.add(new Item("Ring", 1, 500, "Epic")));      // true
bag.display();
// Expected: 3 items listed with indices 0,1,2

// === Test 3: add null, add full ===
System.out.println(bag.add(null));   // false
bag.add(new Item("Potion", 2, 25, "Common"));
bag.add(new Item("Wand", 3, 200, "Rare"));
System.out.println(bag.isFull());    // true
System.out.println(bag.add(new Item("Extra", 1, 1, "Common"))); // false — full!

// === Test 4: remove by index ===
Item removed = bag.remove(0);
System.out.println(removed.getName());  // "Steel Sword"
bag.display();
// Expected: 4 items, indices 0-3. Old index 0 (sword) is gone, everything shifted.

// === Test 5: removeByName ===
Item pot = bag.removeByName("Potion");
System.out.println(pot.getName());      // "Potion"
System.out.println(bag.contains("Potion")); // false

// === Test 6: totals + rarity count ===
bag.display();  // should be 3 items now
System.out.println(bag.getTotalValue());    // sum of remaining values
System.out.println(bag.getTotalWeight());    // sum of remaining weights
System.out.println(bag.countByRarity("Rare"));   // count of Rare items
System.out.println(bag.countByRarity("Legendary")); // 0

// === Test 7: empty inventory ===
FixedInventory empty = new FixedInventory(3);
empty.display();                // "(empty)"
System.out.println(empty.getTotalValue());   // 0
System.out.println(empty.contains("Anything")); // false
