High Score Table — Full Walkthrough

This is the program your teacher builds live. It uses arrays to store scores, find the best one, sort entries, and add new scores. Use this page as a reference.

The Goal

Store player initials and scores in parallel arrays. Find the highest score, sort the leaderboard, and allow inserting new entries.

=== HIGH SCORES ===
1. AAA — 9500
2. BBB — 7200
3. CCC — 5100
4. DDD — 3400
5. EEE — 1200
---
Top score: AAA (9500)
Average: 5280

Step 1: Declare Parallel Arrays

String[] players = new String[5];
int[] scores = new int[5];

// Populate with sample data
players[0] = "AAA";  scores[0] = 9500;
players[1] = "BBB";  scores[1] = 7200;
players[2] = "CCC";  scores[2] = 5100;
players[3] = "DDD";  scores[3] = 3400;
players[4] = "EEE";  scores[4] = 1200;

Parallel arrays: players[i] and scores[i] belong to the same entry. They stay linked by index. This is simpler than a class for small problems, though you could also make a HighScoreEntry class.

Step 2: Find the Top Score

int maxScore = scores[0];
int maxIndex = 0;

for (int i = 1; i < scores.length; i++) {
    if (scores[i] > maxScore) {
        maxScore = scores[i];
        maxIndex = i;
    }
}
System.out.println("Top score: " + players[maxIndex] + " (" + maxScore + ")");

Track the index, not just the value. If you only track maxScore, you can't print the player's name. Track the index and you get both for free.

Step 3: Calculate Average

int sum = 0;
for (int i = 0; i < scores.length; i++) {
    sum += scores[i];
}
double average = (double) sum / scores.length;
System.out.println("Average: " + (int) average);

Cast to double before dividing. sum / scores.length is integer division — 26400/5 = 5280 is fine here, but if average had a decimal, it would be lost. Cast to double first to be safe.

Step 4: Insert a New Score (Sorted Order)

// Given newPlayer and newScore, insert while keeping sorted (highest first)
String newPlayer = "FFF";
int newScore = 6300;

// Find the insertion point
int insertAt = 0;
while (insertAt < scores.length && scores[insertAt] > newScore) {
    insertAt++;
}

// Shift everything down from insertAt onward
for (int i = scores.length - 1; i > insertAt; i--) {
    players[i] = players[i - 1];
    scores[i] = scores[i - 1];
}

// Insert
players[insertAt] = newPlayer;
scores[insertAt] = newScore;

// Note: the last entry (EEE, 1200) falls off the end

The Full Program

public class HighScoreTable {
    public static void main(String[] args) {
        String[] players = {"AAA", "BBB", "CCC", "DDD", "EEE"};
        int[] scores = {9500, 7200, 5100, 3400, 1200};

        // Display leaderboard
        System.out.println("=== HIGH SCORES ===");
        for (int i = 0; i < players.length; i++) {
            System.out.println((i + 1) + ". " + players[i] + " — " + scores[i]);
        }

        // Top score
        int maxIndex = 0;
        for (int i = 1; i < scores.length; i++) {
            if (scores[i] > scores[maxIndex]) maxIndex = i;
        }
        System.out.println("Top score: " + players[maxIndex]
                           + " (" + scores[maxIndex] + ")");

        // Average
        int sum = 0;
        for (int s : scores) sum += s;
        System.out.println("Average: " + (sum / scores.length));
    }
}

Extending the Program

  • Sort the entire table (bubble sort or selection sort).
  • Read new scores from user input with Scanner.
  • Write to a file so scores persist between runs (AP extension).

How This Connects to Your RPG

Your inventory system uses the same array patterns. The top-score search is identical to finding the most valuable item in inventory. The insert-with-shift is the same pattern as adding items to a sorted inventory.
