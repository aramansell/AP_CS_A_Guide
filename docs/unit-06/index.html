Unit 6: Arrays

In Unit 5, you learned to create individual objects. But games don't have one enemy, one item, or one spell. They have collections. A party of four heroes. An inventory of twenty items. A deck of fifty-two cards. A high score table of ten entries.

This unit introduces the array — Java's first collection type. Fixed-size, fast, and the foundation for everything that comes after.

Concepts

TopicWhat you'll learn
Array Basics Declaration, initialization, indexing. The zero-index rule.
Array Traversal Looping through arrays. Sum, average, find, count, accumulate.
Array Algorithms Max, min, shift, reverse. The standard toolkit.

RPG Thread

Your inventory stops being a handful of variables and becomes an Item[]. Your combat stops being one-on-one and becomes Monster[]. In Unit 7, you swap to ArrayList for dynamic sizing. In Unit 8, arrays get a second dimension for dungeon maps. The Card deck you build here feeds into card-based mini-games later.

Prerequisites

Before starting this unit, you should be comfortable with:

  • Writing classes, constructors, and methods: Unit 5
  • Loops (while, for, nested): Unit 4
  • Boolean expressions and if statements: Unit 3

Assessment

  • Part C: Write code — implement an array algorithm from a specification
