4.5c — String Analysis with Loops

Type: Expand Application. You've printed shapes with nested loops — now apply loops to something practical: tearing apart Strings character by character.

Expand Application ~25 min

The Goal

Build a String Analyzer that takes a word or sentence and counts vowels, reverses the text, and checks for palindromes — all by looping over individual characters.

Enter a word: racecar
Length: 7
Vowels: 3
Reversed: racecar
Palindrome? YES!

Build It: String Analyzer

Create StringAnalyzer.java. Start with this skeleton:

import java.util.Scanner;

public class StringAnalyzer {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        System.out.print("Enter a word: ");
        String word = input.nextLine();

        System.out.println("Length: " + word.length());

        // TASK 1: Count vowels
        int vowels = 0;
        for (int i = 0; i < word.length(); i++) {
            char c = word.charAt(i);
            // YOUR CODE: check if c is a vowel
        }
        System.out.println("Vowels: " + vowels);
    }
}

Tinker Challenge — Reverse It

Add a loop that builds the reversed version of the word. Start with an empty String: String reversed = ""; Then loop from the END of the word to the beginning, adding each character. Don't look up the formula — think about what index the last character is at.

Do NOT use StringBuilder.reverse(). Build it character by character. You'll understand loops better.

Bug Hunt

When you loop from the end to the beginning, what index do you start at? Hint: word.length() is 7 for "racecar", but the last character is at index 6. If your loop starts at word.length(), you get a StringIndexOutOfBoundsException. Why?

Palindrome Check

Now that you have the reversed string, checking for a palindrome is trivial:

if (word.equals(reversed)) {
    System.out.println("Palindrome? YES!");
} else {
    System.out.println("Palindrome? No.");
}

But wait — try "Racecar" with a capital R. Does it still work? Fix it so case doesn't matter.

Socratic Checkpoint

  1. Why does word.charAt(word.length()) crash but word.charAt(word.length() - 1) doesn't?
  2. Compare your two loops: the forward loop (counting vowels) and the backward loop (reversing). What's the same? What single thing changes between them?
  3. How would you count only LOWERCASE vowels? What changes?

Extend It (pick one)

  • Count consonants too. Print a vowel-to-consonant ratio.
  • Handle full sentences with spaces: ignore spaces when checking for palindromes ("a man a plan a canal panama").
  • Count how many times each letter appears (frequency analysis). You'll need a way to track 26 counts.
