1.7b — Dice Roller: statRoll()

RPG — Part 2 ~15 min

Part 3: statRoll() — Generate a Single D&D-Style RPG Stat

In many RPGs, character stats are generated by rolling 3 six-sided dice (3d6) and summing the results. Build a method that does this.

Requirements:

  • Method signature: public static int statRoll()
  • Generates three d6 rolls and returns the sum
  • This is a convenience method — it should call the roll() method you already wrote, not repeat the Math.random() logic.

Socratic Checkpoint

Why call roll(6) three times instead of writing (int)(Math.random() * 6) + 1 three times? What happens if you later discover a bug in your random formula — how many places would you need to fix?

Acceptance Test:

for (int i = 0; i < 6; i++) {
    System.out.println("Stat " + (i+1) + ": " + Dice.statRoll());
}

Expected: six numbers, each between 3 and 18.

Tinker: d20 Check

RPGs also use d20 ability checks. Add a method:

public static int d20Check()

This rolls a d20 and prints whether it's: Natural 20 (critical success), Natural 1 (critical failure), or just a normal number. You'll need an if/else chain.
