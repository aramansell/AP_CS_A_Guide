9.1b — Writing Subclasses: Extend and Customize

Basic Application ~20 min

Build: Character Hierarchy

Yesterday you explored IS-A relationships. Today you'll build the hierarchy: a base Character class and subclasses that inherit from it.

Create Character (parent):

  1. Protected fields: String name, int hp, int attack.
  2. Constructor takes name, hp, attack.
  3. Methods: void takeDamage(int dmg), void attack(Character target).

Create Warrior extends Character:

  1. Additional field: int armor.
  2. Constructor: calls super(name, hp, attack), then sets armor.
  3. Override takeDamage(): reduce damage by armor first (minimum 0 damage).

Create Mage extends Character:

  1. Additional field: int mana.
  2. Override attack(): if mana >= 10, deal double damage and spend 10 mana. Otherwise normal attack.
