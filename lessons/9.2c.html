9.2c — Elemental Combat: Override Damage by Type

Expand Application ~30 min

Build: Elemental Damage System

Many RPGs have elemental weaknesses: fire beats ice, ice beats water, etc. Each damage type calculates differently — perfect for method overriding.

Create Attack (abstract parent):

  1. Field: String name, int power.
  2. Abstract method: int calculateDamage(String targetElement).

Create concrete subclasses:

  1. FireAttack extends Attack: against "ice" target → 2× damage. Against "water" → 0.5×. Otherwise normal.
  2. IceAttack extends Attack: against "fire" target → 2×. Against "ice" → 0.5×.
  3. PhysicalAttack extends Attack: normal damage to all. No weakness, no resistance.

Store all 3 attacks in an Attack[] array. Loop through them and test against "ice", "water", and "fire" targets. Print each result.
