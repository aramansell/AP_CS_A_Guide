10.2b — Recursive Traversal: Reverse, Palindrome, and Sum

Basic Application ~20 min

Build: Recursive String and Array Tools

Yesterday you saw how recursion traverses strings and arrays. Today you'll write the classic recursive traversal problems — each one builds on the same pattern.

Create a class called RecursiveTools:

  1. Reverse a String String reverse(String s):
    • Base case: if length <= 1, return s
    • Recursive: last character + reverse(the rest)
  2. Palindrome Check boolean isPalindrome(String s):
    • Base cases: if length <= 1 → true. If first != last → false.
    • Recursive: check the substring without first and last characters
  3. Sum Array int sum(int[] arr, int index):
    • Base case: index >= arr.length → return 0
    • Recursive: arr[index] + sum(arr, index+1)

Trace isPalindrome("racecar")

Write out every recursive call. At what call does the base case fire? How many total calls are made?
