3.2b — Boolean Logic: Combining Conditions in Practice

Basic Application ~20 min

Build: Access Checker

Many programs check multiple conditions at once. Create a class called AccessChecker that decides whether a player can enter a restricted area.

  1. Declare three variables:
    • int playerLevel — set to 5
    • boolean hasKey — set to false
    • boolean isAdmin — set to false
  2. Use && to check: playerLevel >= 3 AND hasKey. Print "Access granted" or "Access denied".
  3. Now modify the condition using ||: allow access if either the player has a key OR is an admin. Print the result.
  4. Add a third check using !: deny access if the player is NOT level 1 and NOT an admin. Print the result.

Truth Table Check

Before running: what output do you expect for level=1, no key, not admin? Trace through each of your three checks by hand. Run the program. Did it match your prediction?
