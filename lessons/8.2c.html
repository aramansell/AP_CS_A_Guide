8.2c — Battle Grid: Range and Area Attacks

Expand Application ~30 min

Build: BattleGrid Scanner

Tactical RPGs use a grid where every tile has a unit or is empty. Area attacks affect tiles in a pattern — a cross, a square, a diamond. 2D traversal patterns make this possible.

Create a class called BattleGrid with char[][] grid = new char[6][6], initialized to '.' (empty). Place 'P' for player at (2,2) and 'E' enemies at (1,2), (3,2), (2,1), (2,3).

  1. void render(): print the grid with coordinate labels.
  2. boolean isInRange(int row, int col, int range): return true if the tile is within range steps from the player using Manhattan distance: |row - playerRow| + |col - playerCol| <= range.
  3. ArrayList<String> getTargetsInRange(int range): return coordinates of all enemies within range.
  4. void attackCross(): mark all tiles in a cross pattern (same row OR same column as player) with 'X'.
