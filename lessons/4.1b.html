4.1b — while Loops: Build a Guessing Game

Basic Application ~20 min

Build: Guess the Number

Yesterday you explored how while loops repeat code until a condition becomes false. Today you'll use that pattern to build the classic number guessing game.

Create a class called GuessingGame:

  1. Generate a secret number: int secret = (int)(Math.random() * 100) + 1;
  2. Hardcode a guess (try several values).
  3. Use a while loop that keeps running while guess != secret.
  4. Inside the loop: print "Too high!" or "Too low!" based on the guess.
  5. Count how many guesses it took.

Infinite Loop Trap

What happens if you forget to update guess inside the loop? Try it (be ready to stop your program). This is the most common while-loop bug. How would you prevent it?
