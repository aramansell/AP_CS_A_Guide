8.1b — 2D Arrays: Read, Write, and Navigate Coordinates

Basic Application ~20 min

Build: Grid Editor

Yesterday you saw that 2D arrays use [row][col] indexing. Today you'll write code that reads, writes, and validates grid positions.

Create a class called GridEditor with int[][] grid = new int[5][5]:

  1. Fill the grid so that grid[row][col] = row * col.
  2. Print the grid with row and column headers.
  3. Method int getValue(int row, int col): return the value. If the coordinates are out of bounds, return -1.
  4. Method void setValue(int row, int col, int value): set the value if in bounds, print an error otherwise.

Boundary Check

Try getValue(5, 0) and getValue(0, 5). Which index is out of bounds? How do you determine the number of rows and columns dynamically (hint: grid.length and grid[0].length)?
