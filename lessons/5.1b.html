5.1b — Your First Class: Write It Yourself

Basic Application ~20 min

Build: Item Class

Yesterday you dissected the anatomy of a class. Today you'll write one from scratch. Create a class called Item (in its own file).

  1. Give it three instance variables: String name, int weight, int value.
  2. Make all instance variables private.
  3. Write a constructor that takes all three values.
  4. Write a void display() method that prints the item's stats in a formatted line.

Test It

In a separate ItemTester class with a main method:

  1. Create three Item objects: "Sword" (weight 8, value 25), "Potion" (weight 1, value 10), "Shield" (weight 12, value 40).
  2. Call display() on each.
  3. What happens if you try to access sword.name directly from ItemTester? Why?
