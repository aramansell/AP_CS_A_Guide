8.1a — 2D Array Basics: Grids, Not Lines

Discovery Lab ~25 min

Every array you've used so far has been a line. But dungeon maps are grids. Chess boards have rows and columns. Screens are pixels arranged in width and height. A 2D array is an array of arrays — a grid where you index by [row][col].

Experiment: Your First Grid

Create GridIntro.java:

// 3 rows, 4 columns
int[][] grid = new int[3][4];

// Fill with row * 10 + col
for (int row = 0; row < grid.length; row++) {
    for (int col = 0; col < grid[row].length; col++) {
        grid[row][col] = row * 10 + col;
        System.out.print(grid[row][col] + " ");
    }
    System.out.println();  // newline after each row
}

Run it. Then swap the loops — put the col loop outside and row loop inside. What happens to the output? This is the difference between row-major and column-major traversal.

Experiment: Three Ways to Initialize

// Way 1: Literal
int[][] a = {{1,2,3}, {4,5,6}, {7,8,9}};

// Way 2: new with dimensions
int[][] b = new int[3][4];  // 3 rows, 4 cols

// Way 3: Ragged — every row different length
int[][] c = new int[3][];
c[0] = new int[2];  // row 0 has 2 columns
c[1] = new int[5];  // row 1 has 5 columns
c[2] = new int[3];  // row 2 has 3 columns

For each: print grid.length and grid[0].length. What does each one give you? Now try c[1].length — does it match c[0].length?

Row vs Column Confusion: grid.length is the number of ROWS. grid[0].length is the number of COLUMNS in row 0. The AP exam loves to swap these. Read the 2D Array Basics docs and memorize: first bracket = row, second = column.

Socratic Checkpoint

  1. If grid = new int[5][8], what is grid.length? What is grid[0].length? What is the last valid index for rows? For columns?
  2. In a ragged array, can you assume grid[0].length works for every row? When would it break?
  3. Read the docs section on how 2D arrays are stored in memory. Why is a 2D array really an array of references?

Checkpoint — before moving on, you must be able to:

  • Declare and initialize 2D arrays (literal, new, ragged)
  • Access elements with [row][col] syntax
  • Distinguish row count (grid.length) from column count (grid[r].length)
  • Explain what a ragged array is
