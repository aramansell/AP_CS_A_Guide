7.1b — ArrayList Methods: Add, Remove, and Find

Basic Application ~20 min

Build: Spellbook Manager

Yesterday you saw how ArrayList grows and shrinks automatically. Today you'll use the key methods — add, remove, get, set, contains — to manage a wizard's spellbook.

Create a class called Spellbook with an ArrayList<String> spells:

  1. Add starting spells: "Fireball", "Heal", "Shield".
  2. Add "Lightning Bolt" at index 1 (shifting existing spells).
  3. Remove "Shield" using remove().
  4. Use contains() to check if "Fireball" is still there.
  5. Use set() to upgrade "Fireball" to "Inferno".
  6. Get the spell at index 2. Print it.
  7. Print spells.size() after each operation.

Observe Index Shifts

After removing an element from the middle, what happens to the indexes of elements that were after it? After inserting at index 1, what happens to the element that was at index 1?
