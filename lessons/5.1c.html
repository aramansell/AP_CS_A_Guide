5.1c — RPG Character Sheet: Build Your Hero

Expand Application ~30 min

Build: CharacterSheet Class

Every RPG has a character sheet: name, class, level, HP, MP, and stats. You'll build the Java class that represents one.

Create a class called CharacterSheet:

  1. Instance variables (all private):
    • String name
    • String characterClass (Fighter, Wizard, Rogue)
    • int level
    • int hp, maxHp, mp, maxMp
    • int strength, dexterity, intelligence
  2. Constructor: takes name and characterClass. Calculates stats based on class:
    • Fighter: str=16, dex=10, int=8, hp=40, mp=10
    • Wizard: str=8, dex=10, int=16, hp=20, mp=40
    • Rogue: str=10, dex=16, int=10, hp=25, mp=15
  3. Method void printSheet() that displays all stats in a formatted character sheet.

Think: Where Should the Logic Go?

You could put the stat-calculation logic in the constructor, or in a separate method. Which makes more sense? What if you later add a "levelUp()" method — where would the new stats be calculated?
