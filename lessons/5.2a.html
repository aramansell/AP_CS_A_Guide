5.2a — Constructors: Building Objects from Scratch

Discovery Lab ~25 min

Yesterday you saw new Dice(6) and it just worked. Today you figure out what actually happens when you call new. The constructor runs, initializes instance variables, and returns a ready-to-use object.

Experiment: Constructor Chaos

Create Weapon.java with two constructors:

public class Weapon {
    private String name;
    private int damage;

    public Weapon(String n, int d) {
        name = n;
        damage = d;
    }

    public Weapon() {
        name = "Fists";
        damage = 1;
    }

    public int getDamage() { return damage; }
    public String getName() { return name; }
}

Test both constructors:

Weapon sword = new Weapon("Longsword", 8);
Weapon fists = new Weapon();

System.out.println(sword.getName() + ": " + sword.getDamage());
System.out.println(fists.getName() + ": " + fists.getDamage());

Now try: Weapon w = new Weapon("Dagger"); — does it compile? Why not?

Shadowing Trap: In the first constructor, the parameters are named n and d. What if you named them name and damage? Read the docs on constructors and the this keyword to find out. Then try it — your Weapon will silently break.

Build: Monster Constructor

Create Monster.java with a constructor that takes name, hp, AC, and attackBonus. Add a toString() method that returns a formatted string like "Goblin [HP:10 AC:12 ATK:+3]". Test with three different monsters.

Socratic Checkpoint

  1. You wrote two Weapon constructors. How does Java know which one to call? What's the rule?
  2. If you delete the no-argument constructor and only keep the one with parameters, what happens to new Weapon()? Why?
  3. The constructors docs mention a default constructor. When does Java give you one for free, and when does it not?

Checkpoint — before moving on, you must be able to:

  • Write a constructor that initializes all instance variables
  • Write multiple constructors (overloading)
  • Use this to disambiguate parameters from instance variables
  • Explain when the default constructor is provided
