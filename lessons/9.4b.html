9.4b — Implementing Interfaces: Multiple Behaviors

Basic Application ~20 min

Build: Multi-Ability Character

Yesterday you saw that interfaces define contracts. A class can implement multiple interfaces — something inheritance alone can't do. You'll build a character that can attack AND heal AND defend.

Create three interfaces:

  1. Attacker: method int attack()
  2. Healer: method int heal()
  3. Defender: method int defend(int incomingDamage)

Create Paladin that implements ALL THREE:

  1. attack(): deals 15 damage.
  2. heal(): restores 10 HP.
  3. defend(int dmg): reduces damage by 8.

Create Priest that implements Healer and Defender only (no attack).

Test by storing a Paladin in each interface type — what methods are available?
