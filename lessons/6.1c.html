6.1c — RPG Inventory: Arrays for Your Loot

Expand Application ~30 min

Build: Item Inventory

RPG inventories have a fixed number of slots. Items can be added if there's room, removed if they exist, and displayed. This is a perfect use case for an array with a counter.

Create a class called Inventory:

  1. Declare String[] slots = new String[8] (8 inventory slots).
  2. Declare int itemCount = 0 to track how many slots are filled.
  3. Method boolean addItem(String itemName): add to the first empty slot. Return false if full.
  4. Method boolean removeItem(String itemName): find the item and set its slot to null. Return false if not found. Compact the array by shifting items left.
  5. Method void display(): print all slots that aren't null.

Test Full Inventory

Add 8 items, then try to add a 9th. Does it correctly refuse? Remove an item from the middle, then display. Are there gaps? Should there be?
