6.2b — Traversal Patterns: Sum, Count, and Filter

Basic Application ~20 min

Build: Data Analyzer

Three traversal patterns appear in every array-based program: sum, count, and filter. You'll implement all three against a dataset.

Create a class called DataAnalyzer. Start with this array:

int[] data = {45, 12, 89, 33, 67, 21, 94, 50, 18, 76};
  1. Sum: compute the total of all values.
  2. Count: count how many values are above 50.
  3. Filter: create a new array containing only the even values. Hint: you need two traversals — one to count evens, one to fill the array.

Pattern Recognition

Look at the loop structure for all three tasks. What's the same? What's different? Can you write a single traversal that does all three at once?
