2.1c — RPG Character Creator: Objects Are Your Heroes

Expand Application ~30 min

Build: Character Sheets as Objects

Every RPG character is an object. Their name, HP, strength, and inventory are instance fields. Their methods — attack, defend, heal — operate on that data. You'll create character objects and make them interact.

Create a class called Hero:

  1. Instance fields: String name, int hp, int maxHp, int attackPower.
  2. Constructor: takes name, hp, attackPower. Sets maxHp = hp.
  3. Method void attack(Hero target): target.hp decreases by attackPower. Print "[name] attacks [target.name] for [attackPower] damage!"
  4. Method void heal(int amount): hp increases (capped at maxHp).
  5. Method boolean isAlive(): return hp > 0.
  6. Method void printStatus(): display name, HP, and alive status.

In your main method: create two heroes, have them attack each other in turns until one falls. Print status after each attack.
