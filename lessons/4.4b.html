4.4b — Nested Loops: Build Useful Grids

Basic Application ~20 min

Build: Multiplication Table

Yesterday you saw how nested loops create grids. Today you'll build something useful: a multiplication table that displays all products from 1×1 through 12×12.

Create a class called MultiplicationTable:

  1. Use nested for loops: outer loop for rows (1 to 12), inner loop for columns (1 to 12).
  2. Print each product aligned in columns. Use System.out.printf("%4d", row * col) for alignment.
  3. After each inner loop, print a newline to start the next row.

Observe

Which executes more often — the outer loop body or the inner loop body? Count the total number of print statements. If you swapped the loop variables (i becomes column, j becomes row), would the table look different?
