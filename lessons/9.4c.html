9.4c — Combat Abilities Engine: Interface-Driven Combat

Expand Application ~30 min

Build: Ability System

Professional RPGs use an interface for all combat actions. "Attack", "Heal", "Fireball", "Shield Block" — they all conform to the same contract. This lets you build an ability bar that works with anything.

Create interface CombatAction:

  1. String getName()
  2. int getPower()
  3. String getTargetType() — returns "enemy", "ally", or "self"
  4. String execute() — returns a description of what happened

Create at least 4 implementations:

  1. Slash: power=12, target=enemy
  2. Fireball: power=25, target=enemy
  3. HealingLight: power=15, target=ally
  4. DefensiveStance: power=0, target=self — execute() returns "Defense doubled for 2 turns!"

Store all 4 in a CombatAction[] array. Print each action's name, power, target, and execution result.
