3.5b — Number Guessing Game (Build Together)

Type: Basic Application. The teacher builds, you follow along. This pulls together everything from Unit 3: if/else, boolean logic, and relational operators — plus Scanner from Unit 2.

Basic Application ~35 min

The Goal

Build a game that picks a random number (1–100) and lets the player guess until they get it right. After each guess, print "too high" or "too low."

I'm thinking of a number from 1 to 100.
Your guess: 50
Too high!
Your guess: 25
Too low!
Your guess: 37
Correct! It took 3 guesses.

Iteration 1: One Guess Only

Create NumberGuessing.java. Start with a single round — no loop yet. Get the skeleton working.

import java.util.Scanner;

public class NumberGuessing {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        int secret = (int) (Math.random() * 100) + 1;

        System.out.println("I'm thinking of a number from 1 to 100.");
        System.out.print("Your guess: ");
        int guess = input.nextInt();

        System.out.println("Secret was: " + secret);
        System.out.println("You guessed: " + guess);
    }
}

Run it. Confirm that secret changes every run.

Socratic Checkpoint

  1. Why does secret need to be an int and not a double?
  2. What does + 1 do in the random formula? What happens if you remove it?

Iteration 2: Add the if/else Chain

Compare the guess to the secret. Add this after reading the guess:

// REPLACE the last two println lines with this:
if (guess == secret) {
    System.out.println("Correct!");
} else {
    System.out.println("Wrong!");
}

Tinker Challenge

Replace the "Wrong!" message with something more helpful. Use the Number Guessing walkthrough to see how. The game should say whether the guess was too high or too low.

Iteration 3: The while-Loop Trap

Now make the game repeat until the player guesses correctly. Add a while loop around the guess+if/else. But here's the broken version:

Scanner input = new Scanner(System.in);
int secret = (int) (Math.random() * 100) + 1;
int guess = 0;

System.out.println("I'm thinking of a number from 1 to 100.");

while (guess != secret) {
    System.out.print("Your guess: ");
    guess = input.nextInt();
    // compare here...
}
System.out.println("Correct! It took X guesses.");

Bug Hunt

Copy this code. Run it. The game works, but something is missing. How many guesses did the player make? The program never tells them. Add a counter variable. Where does it go? When do you increment it? What's the initial value?

Extend It (pick one)

  • Reverse the game: the player picks a number, the computer guesses using binary search.
  • Add a "play again?" prompt that resets the game.
  • Give the player a difficulty choice: easy (1–50), medium (1–100), hard (1–500).

Checkpoint — before moving on, you must be able to:

  • Explain why guess must be initialized before the while loop condition
  • Trace the loop for a specific sequence of guesses
  • Explain the difference between == and != in the loop condition
