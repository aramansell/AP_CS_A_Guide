6.3b — Array Algorithms: Search and Insert in an Ordered List

Basic Application ~20 min

Build: High Score Table

High score tables stay sorted. New scores must be inserted at the right position and everything below shifts down. You'll implement this classic algorithm.

Create a class called HighScoreTable:

  1. Declare int[] scores = new int[10] — the table holds up to 10 scores, kept in descending order.
  2. Initialize with: 950, 820, 780, 650, 500, 0, 0, 0, 0, 0 (5 scores, 5 empty). Track int count = 5.
  3. Method boolean addScore(int newScore):
    • Find where it belongs (shift right all smaller scores).
    • Insert it. Increment count.
    • If table is full (count==10), drop the lowest score.
  4. Method void display() to print the table.

Test

Add score 900. It should land between 950 and 820. Add 100. It should go after 500. Add 999. It should go at the top. Display after each insert.
