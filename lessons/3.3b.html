3.3b — Multi-Branch Menu: Building a Game Menu

Basic Application ~20 min

Build: Town Menu

Most games have menus: Shop, Inn, Blacksmith, Leave Town. Each choice triggers different behavior. An else-if chain is the perfect tool for this.

Create a class called TownMenu. Hardcode an int choice (try 1 through 5, one at a time).

  1. choice == 1 → Print "You enter the shop. A merchant greets you."
  2. choice == 2 → Print "You rest at the inn. HP restored!"
  3. choice == 3 → Print "The blacksmith sharpens your sword. +2 attack."
  4. choice == 4 → Print "You leave town. Safe travels!"
  5. Anything else → Print "Invalid choice. Try again."

Trace Challenge

Swap two of the conditions — move choice 4 above choice 1. Run with choice=4. Does anything change? What if choice=1? When does order NOT matter in an else-if chain?
