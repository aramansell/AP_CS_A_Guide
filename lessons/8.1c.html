8.1c — Tile Map Editor: Paint Your Dungeon

Expand Application ~30 min

Build: TileMap Class

Tile-based games use a 2D array of integers where each number represents a different tile type: 0=floor, 1=wall, 2=door, 3=chest, 4=enemy. Build the editor that manipulates this map.

Create a class called TileMap with int[][] map = new int[8][8]:

  1. Initialize: fill borders with 1 (walls), interior with 0 (floor).
  2. Method void setTile(int row, int col, int type): set a tile. Refuse if out of bounds or if it's a border wall.
  3. Method void placeRoom(int startRow, int startCol, int width, int height): clear a rectangular area to floor (0), surrounded by walls (1).
  4. Method void render(): print the map using symbols: '.' for floor, '#' for wall, '+' for door, '$' for chest, 'E' for enemy.

Test: Build a Level

Place a 3x3 room at (2, 2). Place a chest at (3, 3). Place an enemy at (2, 3). Place a door connecting the room to the edge. Render after each operation.
