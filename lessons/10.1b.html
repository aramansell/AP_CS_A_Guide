10.1b — Recursive Patterns: Factorial, Power, and Fibonacci

Basic Application ~20 min

Build: Recursive Math Methods

Yesterday you saw how a method can call itself. Today you'll write the three classic recursive functions. Each one teaches a different pattern.

Create a class called RecursiveMath:

  1. Factorial int factorial(int n):
    • Base case: n <= 1 → return 1
    • Recursive case: n * factorial(n-1)
  2. Power int power(int base, int exp):
    • Base case: exp == 0 → return 1
    • Recursive case: base * power(base, exp-1)
  3. Fibonacci int fib(int n):
    • Base cases: n == 0 → 0, n == 1 → 1
    • Recursive case: fib(n-1) + fib(n-2)

Watch the Performance

Try fib(40). Why is it so slow? Draw the call tree for fib(5) — how many times is fib(2) called? This is the classic recursive performance trap.
