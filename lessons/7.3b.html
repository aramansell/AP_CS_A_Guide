7.3b — Music Playlist (Build Together)

Type: Basic Application. The teacher builds, you follow along. Your first ArrayList — dynamic sizing, add/remove/get, and iteration. No more fixed-size arrays.

Basic Application ~30 min

The Goal

Build a playlist manager using ArrayList. Add songs, remove songs, shuffle the order, and print the full playlist.

=== MY PLAYLIST ===
1. Bohemian Rhapsody - Queen
2. Billie Jean - Michael Jackson
3. Hotel California - Eagles
---
Shuffling...
---
1. Hotel California - Eagles
2. Bohemian Rhapsody - Queen
3. Billie Jean - Michael Jackson

Reference: Music Playlist Walkthrough — full reference.

Iteration 1: Create the Playlist

Create PlaylistManager.java:

import java.util.ArrayList;

public class PlaylistManager {
    public static void main(String[] args) {
        ArrayList<String> songs = new ArrayList<>();

        songs.add("Bohemian Rhapsody - Queen");
        songs.add("Billie Jean - Michael Jackson");
        songs.add("Hotel California - Eagles");

        // Print with numbers
        for (int i = 0; i < songs.size(); i++) {
            System.out.println((i+1) + ". " + songs.get(i));
        }
    }
}

Socratic Checkpoint

  1. What's the difference between songs.size() and songs.length? One works, one doesn't. Why?
  2. What are the angle brackets <String> for? What happens if you leave them off?
  3. Why doesn't ArrayList need a fixed size like arrays do?

Iteration 2: Add and Remove

Add Scanner and let the user manage the playlist:

import java.util.Scanner;

// ... inside main, after the initial songs:
Scanner input = new Scanner(System.in);

while (true) {
    System.out.print("Add, Remove, Print, or Quit? ");
    String cmd = input.nextLine();

    if (cmd.equalsIgnoreCase("add")) {
        System.out.print("Song: ");
        songs.add(input.nextLine());
    } else if (cmd.equalsIgnoreCase("remove")) {
        System.out.print("Number to remove: ");
        int num = input.nextInt();
        input.nextLine();  // buffer trap!
        songs.remove(num - 1);  // why -1?
    } else if (cmd.equalsIgnoreCase("print")) {
        // print loop here
    } else if (cmd.equalsIgnoreCase("quit")) {
        break;
    }
}

Bug Hunt

What happens if the user types "remove 99" when there are only 3 songs? Try it. Then fix it: check that the number is valid before calling remove().

Iteration 3: Shuffle

Add a shuffle command. The algorithm is the same swap pattern you used for Card Deck, but songs.get(i) and songs.set(i, value) replace array bracket access:

// Shuffle: same swap algorithm, ArrayList-style
for (int i = 0; i < songs.size(); i++) {
    int swapIndex = (int) (Math.random() * songs.size());
    String temp = songs.get(i);
    songs.set(i, songs.get(swapIndex));
    songs.set(swapIndex, temp);
}

Checkpoint — your playlist must:

  • Add new songs from user input
  • Remove songs by number (with bounds checking)
  • Shuffle using the swap algorithm
  • Print the playlist with numbers
