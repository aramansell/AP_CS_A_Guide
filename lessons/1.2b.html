1.2b — Putting It Together: StatBlock

Discovery Lab ~20 min

Now you combine everything from the last three pages: variables, math, random, and printing. Create a file called StatBlock.java.

Your Mission:

  1. Declare variables: strength, dexterity, constitution, intelligence — all ints with hardcoded values of your choice.
  2. Calculate averageStat — the sum of all four divided by 4. Watch the integer division trap!
  3. Calculate hitPoints = constitution × 5
  4. Print a formatted character sheet with labels using concatenation and tabs

Your output should look something like:

=== CHARACTER STATS ===
STR    15
DEX    12
CON    14
INT    10
------------------------
Average: 12.75
HP:      70

Socratic Checkpoint

When you calculate the average, why can't you just write (str + dex + con + int) / 4? What would happen? Test it. Then fix it.

Tinker: Make It Random

Instead of hardcoding the stats, generate them randomly using Math.random(). Roll 3d6 for each stat (a 3d6 roll gives a range of 3–18).

Now run the program five times. Do you get different characters each time? What's the best stat roll you got? The worst?

Checkpoint — before moving on, you must be able to:

  • Write a program that declares variables, calculates with them, and prints formatted output — all in one file
  • Spot and fix the integer division trap in a real calculation
  • Use tabs to create aligned column output
