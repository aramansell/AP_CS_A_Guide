3.5a — De Morgan's Laws: Flipping Conditions

Discovery Lab ~20 min

You have a condition. It works. Then someone asks: "Can you rewrite that WITHOUT the ! on the outside?" That's what De Morgan's Laws let you do — flip conditions into equivalent forms.

The Two Laws

Law 1:  !(A && B)  →  !A || !B
Law 2:  !(A || B)  →  !A && !B

In words: NOT (both are true) means at least one is false. NOT (either is true) means both are false.

Experiment: Prove It

Create DeMorganTest.java. Pick two booleans (try TT, TF, FT, FF). For each pair, print BOTH sides of Law 1 and verify they give the same result. Then do the same for Law 2. Build a truth table proving both laws hold.

Socratic Checkpoint

  1. Read the De Morgan's Laws docs. Look at the "Why This Matters" section. Which version is clearer to you: !(score < 0 || score > 100) or score >= 0 && score <= 100? Why?
  2. Now find the Bug Hunt in the docs. The student wanted to check for extreme d20 rolls (1 or 20). They started with !(roll > 1 && roll < 20) which works, then "removed the !" and got roll < 1 && roll > 20. Why will that NEVER be true? How SHOULD they have applied De Morgan's Law?

Apply: Rewrite These

Each of the following uses an outer !. Rewrite it without the ! using De Morgan's Laws:

  1. !(hp <= 0 || armorBroken) — when would this be true?
  2. !(enemyAlive && hasAmmo) — when can't you attack?
  3. !(score < 0 || score > 100) — what does this check?

Write your answers down on paper first. Then write code to verify both versions give the same result for multiple input values.

Apply: Combat Validation

Your combat system needs to validate whether a player can attack. A player CAN attack when they are alive AND have a weapon AND the enemy is alive. Write this condition.

Now the game needs to print "You cannot attack!" when the player CANNOT attack. You could write: 

if (!(playerAlive && hasWeapon && enemyAlive)) {
    System.out.println("You cannot attack!");
}

Apply De Morgan's Law to rewrite this without the outer !. Which version is more readable? Which would be better for printing a specific reason (e.g., "You're dead!" vs "You have no weapon!")?

Checkpoint — before moving on, you must be able to:

  • State both of De Morgan's Laws
  • Rewrite a negated compound condition without the outer !
  • Verify that both forms produce the same truth table
  • Choose the more readable form for a given situation
