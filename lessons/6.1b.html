6.1b — Array Indexing: Read, Write, and Don't Crash

Basic Application ~20 min

Build: Score Tracker

Yesterday you learned that arrays are indexed from 0 and their length is fixed. Today you'll build a score tracker that stores the last 5 game scores and computes stats from them.

Create a class called ScoreTracker:

  1. Declare int[] scores = new int[5].
  2. Fill it with values: 85, 92, 78, 96, 88.
  3. Print the first score (scores[0]) and the last score (scores[4]).
  4. Calculate and print the average score.
  5. Find and print the highest score.
  6. Replace the lowest score with 100. Print the array before and after.

Boundary Trap

Try to access scores[5] or scores[-1]. What error do you get? This is the most common array bug. How do you figure out the last valid index?
