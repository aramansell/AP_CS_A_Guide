3.1b — Conditional Scoring: Apply Your if Statements

Basic Application ~20 min

Build: Score Calculator

Yesterday you explored how if and relational operators let your program make decisions. Today you'll apply that to a scoring system — the kind that shows up in every game.

Create a class called ScoreCalculator with a main method. The program should:

  1. Declare an int score and assign it a value (start with 75).
  2. Use if / else if / else to print a letter grade:
    • 90+ → "A"
    • 80-89 → "B"
    • 70-79 → "C"
    • 60-69 → "D"
    • Below 60 → "F"
  3. Add a second check: if the score is exactly 100, also print "Perfect Score!"

Extend: Bonus Round

Add a boolean hasBonus variable. If hasBonus is true, add 5 points to the score before calculating the letter grade. Test with score = 87 and hasBonus = true. What letter grade do you get now?

Think

What happens if you put the "Perfect Score" check after the letter grade chain? What if you put it before? Try both. Which order makes more sense?
