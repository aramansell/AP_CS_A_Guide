4.1c — Combat Loop: Fight Until Someone Falls

Expand Application ~30 min

Build: Combat Loop

RPG combat isn't one hit and done. It's a loop: hit, hit, hit, until someone's HP reaches zero. You now understand while loops well enough to build this.

Create a class called CombatLoop:

  1. Declare int playerHP = 30 and int enemyHP = 25.
  2. Use a while loop: while (playerHP > 0 && enemyHP > 0).
  3. Inside the loop:
    • Player attacks: enemyHP decreases by a random amount (5-10).
    • Enemy attacks: playerHP decreases by a random amount (3-8).
    • Print the current HP values after each exchange.
  4. After the loop ends, print who won.

Think: Loop Exit Conditions

Your loop condition uses &&. What happens if you use || instead? What does the && guarantee that || wouldn't?
