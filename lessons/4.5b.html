4.5b — Pattern Printer (Build Together)

Type: Basic Application. The teacher builds, you follow along. Nested loops turn abstract loop tracing into visible output. You'll see exactly what an inner loop does relative to its outer loop.

Basic Application ~30 min

The Goal

Print a right triangle of stars using nested for loops.

*
**
***
****
*****

Iteration 1: Single Row First

Before nesting, make sure you can print a single row. Create PatternPrinter.java:

public class PatternPrinter {
    public static void main(String[] args) {
        // Print 5 stars on one line
        for (int i = 0; i < 5; i++) {
            System.out.print("*");
        }
        System.out.println(); // move to next line
    }
}

Run it. You should see ***** on one line.

Socratic Checkpoint

  1. What's the difference between print and println here? Why do we need both?
  2. If you changed the loop to i < 3, how many stars would print?

Iteration 2: Repeat the Row (Broken!)

Wrap the existing code in another for loop. Let's see what happens:

for (int row = 0; row < 5; row++) {
    for (int i = 0; i < 5; i++) {
        System.out.print("*");
    }
    System.out.println();
}

Run it. You get a 5×5 square of stars. That's not a triangle.

Bug Hunt

The inner loop always prints 5 stars — every row is the same length. To make a triangle, row 0 should print 1 star, row 1 should print 2 stars, row 2 should print 3 stars...

What needs to change in the inner loop's condition? The condition currently says i < 5. What variable from the outer loop could replace the 5?

Iteration 3: The Inverted Triangle Trap

Change the inner loop's condition to i < row. Run it. You get:


*
**
***
****

Almost right, but row 0 printed 0 stars. Why?

Tinker Challenge

Fix it so row 0 prints 1 star and row 4 prints 5 stars. The fix is one character. (Hint: should the condition be i < row or i <= row? Or should row start at 1 instead of 0?)

Iteration 4: Different Patterns

Now that you understand how the outer row variable controls the inner loop, try these patterns. For each one, write pseudocode BEFORE you write Java:

*****          *               *****
****           **              ****
***            ***             ***
**             ****            **
*              *****           *

Socratic Checkpoint

  1. For the middle triangle (spaces before stars): how many spaces do you need on each row? What's the relationship between the row number and the number of spaces?
  2. Why does nested loop tracing matter for your RPG? Think about your TileMap: what two things are you looping over to draw every tile?
