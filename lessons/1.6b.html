1.6b — Random Tip Generator

Expand Application ~15 min

Sometimes you want to leave a random tip just for fun. Add a random tip mode.

  1. Replace the hardcoded tipPercent with a random value from 10 to 30 (inclusive).
  2. Print the randomly selected tip percentage so the user can see it.
  3. Run the program multiple times — does it feel unfair? Why might you want a fixed percentage instead?

Tinker: Two Modes

Add a boolean randomMode = true; variable. If it's true, use the random tip percentage. If false, use the hardcoded one. Test both modes.

Socratic Checkpoint

You just combined two concepts — random number generation and the tip calculator — that you learned separately. What other concepts could you combine with the tip calculator? An RPG-themed tip calculator? A tip calculator that prints ASCII art receipts?
